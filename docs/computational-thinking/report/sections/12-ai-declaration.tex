\subsection{AI usage and declaration}

Trong quá trình hoàn thiện đồ án, nhóm đã sử dụng các công cụ AI hỗ trợ sau:
\begin{itemize}
    \item \textbf{Google Gemini 2.0 Flash}: Làm LLM engine sinh nội dung quest trong Quest Maker.
    \item \textbf{Gemini Code Agent / Antigravity}: Hỗ trợ tạo khung code, debug lỗi kiểu dữ liệu và định dạng file cấu hình.
    \item \textbf{BAAI/bge-m3}: Embedding model chạy local để sinh vector database cho tri thức văn hóa.
\end{itemize}
